getUserDetails

查询用户详细信息

作者: May

请求URL

/profile/getUserDetails POST

请求参数 application/x-www-form-urlencoded

参数名 类型 必须 描述
phoneNumber string 是 用户手机号

返回结果

{
	"phoneNum":"string //电话号码",
	"userName":"string //用户名",
	"studentId":"string //学号",
	"dormNum":"int //宿舍号",
	"gender":"int //性别（1-男，0-女）",
	"userScore":"int //信誉分（满分10）",
	"avatar":"string //头像URL"
}
